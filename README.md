# Agent Workflow Frontend

This project provides a simplified workflow builder inspired by **n8n**. It
focuses only on the frontend using React and Vite.

## Features

- Searchable palette to add nodes such as MCP, Database, API and GitHub steps.
- Simple canvas listing nodes as sequential steps.
- Extensible foundation for connecting backend logic in the future.

## Development

```bash
# install dependencies
yarn install

# start local dev server
yarn dev
```

## Linting & Type Checking

```bash
yarn lint
yarn type-check
```
